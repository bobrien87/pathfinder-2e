###  Antigravity Support

This vault includes built-in support from Google Antigravity (AI). This is entirely optional and you can ignore it if you don't want to use it.

#### Workflows
Workflows are used to automate common tasks that you might want an AI to perform. The following workflows are included:
- `/insert` is used to automatically add something to your vault, for example a creature, NPC, location or hazard
- `/import` is used to automatically sync any data that exists in the Foundry VTT module into the vault
- `/convert-sources` is used to convert PDF sources into markdown files, for faster reference

#### Rules
Rules are used to ensure that any notes created using AI maintain a consistent format, look and feel, and are compatible with the plugins that are installed.

#### Skills
Skills are used to allow AI to complete certain tasks, triggered by workflows or other user requests. For example, managing the import from Foundry VTT.
